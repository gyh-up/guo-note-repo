# 问题：环形链表

[哈希表](/classify/algorithm/基础数据结构-哈希表) | [链表](/classify/algorithm/基础数据结构-链表) | [双指针](/classify/algorithm/技巧-双指针) | [相遇问题](/classify/algorithm/数学-相遇问题)

给你一个链表的头节点 head ，判断链表中是否有环。
